Website Overview

This site was created to help visualize various college metrics supplied by www.collegescorecard.ed.gov.

Current data is from calendar year 2017 and is updated annually.

  • Information provided includes:
    • school Name
    • State
    • City
    • In State Tuition
    • Out of State Tuition

This site is built with a PostgreSQL and Flask backend and includes visualizations using Plotly, Leaflet, D3, Google Charts and Fusion Charts. More info on these libraries can be found in the Library References section.
